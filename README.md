# 주제 : 캐릭터 파쿠르 시스템
## 일정 (임시)
- 1주차 : 엔진 기반 준비
    - 씬, 액터, 캐릭터 등 기반 클래스
    - 모델 로드
    - 애니메이션 재생
- 2주차 : 캐릭터 및 애니메이션 시스템
    - 캐릭터 입력 바인딩
    - 애니메이션 시스템 : 이동, 달리기, 점프 등
    - 간단한 씬 세팅
- 3주차 : 파쿠르 애니메이션 적용
    - 장애물 유형화 및 충돌 인식
    - 장애물 넘기, 벽면 매달리기, 벽면 이동
- 4주차 : 프로토타입
    - 전진 방향 내 장애물 식별
    - 장애물 - 캐릭터 간 위치에 따른 장애물 이동 세분화
- 5주차 : 미흡 사항 보완
    - 어색한 부분 다듬기
- 6주차 : 디테일 보정 및 지형 수정
    - 캐릭터 애니메이션 IK 활용 디테일 보정
- 7주차 : 복잡한 지형 내 이동 시, 파쿠르 확인
    - 복잡도 있는 지형 변경
    - 벽-다른 벽 이동 등 세부 테스트 및 보정
- 8주차 : 마무리
    - 테스트 및 오류 수정
    - 미흡 사항 보완
